Sobre o Projeto

Este site foi desenvolvido como um projeto acadêmico com o objetivo de criar uma plataforma moderna para consulta de informações sobre jogos, funcionando como uma espécie de Wikipédia Gamer.

Na plataforma, é possível visualizar detalhes de diversos jogos, como sinopse, desenvolvedora, ano de lançamento, avaliação e muito mais. Além disso, o site conta com funcionalidades administrativas que permitem adicionar, editar e excluir jogos do sistema.

Usuários podem criar uma conta e fazer login para acessar as funcionalidades completas, garantindo uma experiência mais personalizada e segura.

Todo o projeto foi construído exclusivamente com HTML e CSS, demonstrando o poder dessas tecnologias mesmo sem o uso de linguagens de programação de back-end ou banco de dados. Todas as telas, interações e páginas foram desenvolvidas com atenção à responsividade e à usabilidade.

O site é parte de um trabalho prático que tem como foco aplicar conhecimentos de estruturação de páginas, estilização, navegação e experiência do usuário.
